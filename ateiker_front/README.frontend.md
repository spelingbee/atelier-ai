# AtelierAI — Frontend (один файл)

`index.html` — самодостаточный SPA-визард без сборки и без npm. Открывается в любом браузере.

## Как запустить

1. Подними бэкенд (см. README.backend.md): `uvicorn api:app --port 8000`
2. Открой `index.html` двойным кликом, либо раздавай статикой:
   `python3 -m http.server 5500` → http://localhost:5500/index.html
3. В поле **API** внизу левой панели укажи адрес бэкенда (дефолт http://localhost:8000).

CORS на бэкенде уже разрешён (`allow_origins=['*']`).

## Флоу

1. **Фото** — drag&drop или выбор файла → `POST /api/v1/analyze`.
2. **Тип** — показывается результат НС + уверенность; можно исправить вручную (фоллбэк).
3. **Мерки** — слайдеры талия/бёдра/длина + расширенные (прибавки, припуск) → `POST /api/v1/generate`.
4. **Лекало** — SVG встраивается в превью 1:1, кнопки скачивания PDF/SVG.

## Стиль

Собран на Notion-дизайн-системе (светлая/тёмная темы автоматически). Весь CSS вшит в файл.
